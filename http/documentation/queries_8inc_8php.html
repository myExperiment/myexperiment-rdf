Detailed Description

A set of SPARQL queries to support generation of an HTML specification document for a file that neither specifically an OWL ontology or RDF schema.

Version:
beta
Author:
David R Newman

This file provides an array of SPARQL queries that extract classes, properties and relationship thereof for any non-specific schema. The results from tehse queries can then be used to generate an HTML specification document for any non-specific schema.
