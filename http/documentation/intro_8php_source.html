This guide is intended to help explain the basics of SPARQL and give usable examples that will return actual results from <a href="sparql">myExperiment's SPARQL endpoint</a>.</p>
00063 <p>The first section explains how to use the SPARQL endpoint and the unique features of the  myExperiment SPARQL endpoint.</p>
