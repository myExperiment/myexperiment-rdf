Detailed Description

A set of SPARQL queries to support generation of an HTML specification document for any OWL ontology.

Version:
beta
Author:
David R Newman

This file provides an array of SPARQL queries that extract classes, properties and relationship thereof for any OWL ontology. The results from tehse queries can then be used to generate an HTML specification document for any OWL ontology.
