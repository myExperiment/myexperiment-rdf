Detailed Description

Functions used by myExperiment's RDF generator at rdfgen/rdfgencli.php.

Version:
beta
Author:
David R Newman

Functions used by myExperiment's RDF generator at rdfgen/rdfgencli.php. It also requires various other sets of functions and configuration including:

  • connect/myexperiment.inc.php: To setup a connection to the myExperiment database.
  • functions/utility.inc.php: Various generic utility functions.
  • functions/property.inc.php: Functions for generating property/value pairs for myExperiment RDF entities.
  • config/data.inc.php: Specifies how myExperiment RDF entities should be generated from the myExperiment database.
  • functions/sql.inc.php: SQL specific functions.
  • functions/xml.inc.php: XML parsing, generating and examining functions.

Generates RDF/XML for a property by builidng a URI from optionally prepending $datauri and the URL subpath for the entity type before the value for the specified database field.

E.g. 'user_id'=>'&User|mebase:has-announcer' --> <mebase:has-announcer rdf:resource="&lt;datauri&gt;users/&lt;user_id&gt;"/>

Parameters:
$fieldA string containing the name of the database field from which the value is provided.
$propertyA string containing the RDF property name and if required the type of entity for which the URI is being generated.
$valueA string containing the value that makes up the final part of the URI being generated as part of the RDF property.
$msgA string optionally containing comma-separated messages about any special processing for generating the RDF property. In particular NoDataURI which uses $value as the full URI rathering than prepending $datauri or the entity's URL subpath.
Returns:
A string containing the RDF/XML for the specified property including the URI reference to a particular instance.

printFunctionProperty ( $  property,
$  entity,
$  type = "",
$  msg = "" 
)

Generates RDF/XML for a property by building the value of the property by calling a particular function.

E.g. 'requester'=>'&#64;getRequester|mebase:has-requester' --> <mebase:has-requester rdf:resource="&lt;datauri&gt;&lt;returned_by_function_call&gt; (Where &lt:returned_by_function_call&gt; may be something like "users/16."

Parameters:
$propertyA string containing the RDF property name.
$entityAn array containing the results from a database query for the entity for which the RDF/XML property is being generated.
$typeA string containing the type of entity for which the RDF/XML property is being generated.
$msgA string optionally containing comma-separated messages about any special processing for generating the RDF property. In particular NoDataURI which uses whatever is returned from the function as the full URI rathering than prepending $datauri or the entity's URL subpath. Or EncapsulatedObject which encapsulates whatever is returned from the function between start and end tags for the property (rather than assuming it is a URI).
Returns:
A string containing the RDF/XML of the specified property and either a URI reference to an entity or some additional RDF/XML generated by a separate function.
