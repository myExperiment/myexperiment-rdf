setKey ( $  arr,
$  key 
)

Set of the key for each top-level entry in a multidimensional array to the value of one of the fields in the sub-array for that top-level entry.

E.g. $arr = array(0 => array("name" => "", "size" => "small"), 1 => array("name" => "eagle", "size" => "large")); and $key = "name" would return: array("robin" => array("name" => "robin", "size" => "large"), "eagle" => array("name" => "eagle", "size" => "large"));

Parameters:
$arrA multi-dimensional array for which the key to top-level entries is to be set.
$keyThe field in the top-level entry's sub-array thats value should be used as the key for top-level of the array.
