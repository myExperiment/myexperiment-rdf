Detailed Description

RDF for single class/property in myExperiment Ontology.

Version:
beta
Author:
David R Newman

Displays RDF/XML for individual class or property in one of the modules of the myExperiment Ontology. E.g. ontent.php?ontology=contributions&entity=Workflow display the RDF/XML for the Workflow class.
