Send a SPARQL query (and retrieve results) to a specified knowledge base. (This will be refactored when the codebase is redesigned to use 4Store's HTTP interface).

Parameters:
$kbA string containing the knowledge base for which a SPARQL query is to be submitted.
$queryA string containing the SPARQL query to be submitted to a specified knowledge base.
$formatA string containing the format required from the resukts of the SPARQL query, (e.g. sparql, test or json).
$softlimitAn integer containing the soft limit (a 4Store parameter for how much resource to put towards finding results) for the query being executed.
$reasoningA boolean specifying whether reasoning (provided by 4store-reasoner) should be used for the query specified.
Returns:
A string containing the results from the SPARQL query specified in the format also specified.
