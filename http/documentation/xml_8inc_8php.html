Detailed Description

Functions for parsing, generating and examining XML.

Version:
beta
Author:
David R Newman

Functions for parsing, generating and examining XML. Including myExperiment RDF/XML and SPARQL results in application/sparql-results+xml format and XML representations of workflows including Taverna, Taverna 2, Galaxy and RapidMiner.
