Detailed Description

Queries the myExperiment 4Store knowledge base to determine the datatypes of all myExperiment properties.

Version:
beta
Author:
David R Newman

This script use the 4s-query command line utility to perform a SPARQL query of the myExperiment 4Store knowledge base to determine the datatype of each property defined in the myExperiment ontology (that has already been imported into this knowledge base).
